# <Player_evaluation_app>

## サイト概要
### サイトテーマ
プロ野球の試合で、その日の出場選手の活躍度合いを評価して投稿し、他の人の評価なども見れるようなレビューサイト

### テーマを選んだ理由
海外サッカーなどでは選手を評価するようなシステムが存在しますが、その野球バージョンは見たことがありませんでした。
１試合全てを観戦できない人でもこのサイトを閲覧することで、その試合で何が起きてどうなったのか、勝負どころがどこだったのか知ることができます。
また、他の人がどのような視点で試合を見ていたかを知ることでよりコアかつ多角的な視点で野球を見ることができると考え、このテーマにしました。

### ターゲットユーザ
・野球観戦が好きな人
・もっと野球について知りたい人
・試合を観戦できなかった・見逃してしまった人

### 主な利用シーン
・その日の試合について知りたいとき
・他人の意見を知りたいとき
・その日の試合について自分の感想を他人とシェアしたいとき
